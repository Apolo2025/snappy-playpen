# kdenlive-devel snap

This project creates a working snap of `kdenlive`.

This snaps builds git master versions of:
 - Vid.stab
 - Frei0r
 - OpenCV
 - MLT
 - Kdenlive

## Current state

Working features:
 - `kdenlive-devel`

Known issues:
  - Due to snap limitations (acces to D-Bus) currently works only in devmode.
